# Lead Generation System

Built a B2B lead generation system targeting US and Indian universities.

## What it does
- Scrapes contact data for 4 key decision-maker roles per university
- Processes and cleans data using openpyxl
- Integrates Gemini API for intelligent data processing
- Achieved 99% email deliverability

## Tech Stack
Python, BeautifulSoup, openpyxl, Gemini API

## Results
10,000+ verified contacts across 500+ universities
